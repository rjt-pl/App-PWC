=head1 NAME

App::PWC::Rebuild - Walk all challenge-* directories and process data

=head1 SYNOPSIS

    use App::PWC::Rebuild;
    my $rebuild = App::PWC::Rebuild->new( $pwc );

    $rebuild->rebuild;

=head1 DESCRIPTION

This module generates and processes all submissions data for L<App::PWC>.

=head1 CONSTRUCTOR

=head2 new( %options )

Creates the Data object. B<If you are using App::PWC, you do not
need to do call this constructor directly.>

    my $users = App::PWC::Data->new ( base => '/path/to/pwc-club' )

=head1 METHODS

=head2 rebuild

Rebuild all tables. The existing tables will be cleared first.
This function will C<CREATE> the tables if they do not already exist.

The tables themselves are not C<DROP>ped; call C<$rebuild-&gt;drop_tables>
for that.


=head2 drop_tables

This method will C<DROP> all tables in the database. This is not
normally needed, but may be necessary if the schema changes between
versions.


=head1 DIAGNOSTICS

=head2 Errors

This module will C<croak()> on unrecoverable errors, such as filesystem
access.

Otherwise, nonexistent records will return C<undef> as described in the
method.

=head2 Warnings

There are currently no warnings generated by this module. That may change in
the future, and those warnings will be signaled via C<carp()>.
